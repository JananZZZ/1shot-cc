### 标题

### 正文

姐妹们！！！我发现了一个超级好用的东西 😭

事情是这样的，我最近想试试 AI 编程，听说 Claude Code 很厉害，结果一看教程——要装 Node.js、要配 Git、要改 PowerShell 安全设置、还要自己写配置文件……

作为一个看到黑框框就害怕的人，我直接劝退了。

然后朋友给我推荐了这个——**1shot-CC**，说「你就点鼠标就行」。

我抱着试试看的心态……居然真的 5 步装好了！！！👇

**第1步** 🔍 打开软件，自动检查电脑环境
不用你输入任何东西，它自己帮你看了——Node.js 装了没、Git 有没有、安全设置对不对，9 项检查一目了然。

**第2步** 📥 缺什么点一下，自己帮你装
我电脑没有 Node.js 和 Git，点了一下按钮，它就自动从国内镜像下载了，不用科学上网，速度超快。

**第3步** 🤖 一键装好 Claude Code
再点一下，Claude Code 就装好了。什么 npm 命令、PowerShell 策略，它自己全处理了。

**第4步** ⚙️ 选一个 AI 服务商，填上 Key
内置了智谱、DeepSeek、MiniMax、小米 MiMo、通义千问 5 家，选一家把 API Key 贴进去就行。甚至连模型都帮你选好了推荐的，不用自己操心！

**第5步** 🎉 点「开始我的第一次 Coding」
撒花庆祝！然后自动打开 Windows Terminal，直接就能和 Claude Code 对话了。

**重点是：完全免费，所有操作都在你自己的电脑上。**
