# CC-Switch 使用教程

## 什么是 CC-Switch？
开源跨平台桌面应用（GitHub 40K+ Stars），一键管理 Claude Code 的 API 供应商配置。

## 核心功能
- 50+ 供应商预设，一键切换
- 系统托盘快捷切换，Claude Code 热重载
- MCP 服务器跨应用统一管理
- Token 用量统计和成本监控
- 故障转移自动切换

## 快速上手
1. 启动 CC-Switch
2. 从预设列表选择供应商
3. 输入 API Key 并保存
4. 右键系统托盘图标切换

## 安装
- GitHub: github.com/farion1231/cc-switch
- Windows: 下载 .msi 安装
