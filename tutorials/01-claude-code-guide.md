# Claude Code 完全使用指南

## 什么是 Claude Code？
Claude Code 是 Anthropic 推出的命令行 AI 编程助手。在终端（命令窗口）中运行，通过自然语言对话完成代码编写、调试、重构、文件管理等。

## 基本功能
1. **代码生成与编辑** - 自动创建和修改文件
2. **代码理解与导航** - 分析项目结构，解释代码逻辑
3. **调试与修复** - 运行测试分析错误，追踪代码路径
4. **Git 操作** - 提交代码、创建 PR
5. **Shell 命令执行** - 安装依赖、运行脚本

## 常用快捷键
| 快捷键 | 功能 |
|--------|------|
| Ctrl+C | 取消当前操作 |
| Ctrl+D | 退出 Claude Code |
| Ctrl+O | 查看详细思考过程 |
| Alt+T | 切换扩展思考模式 |
| ↑/↓ | 浏览历史命令 |
| Tab | 自动补全 |

## 核心术语
- **Agent**: 子任务执行单元
- **Tool**: 可调用的操作接口（读写文件、运行命令等）
- **Hook**: 事件触发的自动化操作
- **MCP**: 模型上下文协议，连接外部服务
- **Token**: AI 文本处理单位
- **Extended Thinking**: 深度推理模式
