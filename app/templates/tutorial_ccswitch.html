🔀 CC-Switch 使用教程

多服务商一键切换 · 50+ 预设 · 小白友好

CC-Switch 是什么？

CC-Switch 是一个桌面小工具，专门用来管理 Claude Code 的 AI 服务商配置。

简单说：有了它，你今天想用 DeepSeek，明天想试智谱，点一下就切过去了。不用每次去改配置文件，不用记各种 Base URL。

怎么安装？

两种方式：

  • 桌面版（推荐）：在 1shot-CC 向导里一键安装，或去 GitHub 下载 .msi 文件双击安装
  • 命令行版：终端运行 npm install -g cc-switch-cli

快速上手：3 步配好

  1. 打开 CC-Switch — 从开始菜单找，或点右下角任务栏图标
  2. 添加服务商 — 从内置列表里选一家（比如「智谱 GLM」），粘贴你的 API Key
  3. 一键切换 — 选中想用的服务商，点切换。Claude Code 会立刻用上

就这么简单。不用改任何文件，不用敲任何命令。

常见问题

Q：切换服务商后，Claude Code 需要重启吗？
A：不需要。CC-Switch 桌面版支持热切换，Claude Code 会自动用新配置。
Q：为什么切换后 Claude Code 报错？
A：检查三件事：① API Key 有没有复制完整 ② 账户里还有没有余额 ③ 网络能不能连上服务商。
💡 推荐策略：配 2-3 个服务商，开故障转移。日常用 DeepSeek（便宜），关键时刻切智谱（稳定）。
