🔀 CC-Switch 使用教程

多供应商一键切换 · 50+ 预设 · MCP 统一管理

什么是 CC-Switch？

CC-Switch 是一款开源的跨平台桌面应用（GitHub 40K+ Stars），专门用于管理 Claude Code 的 API 供应商配置。它解决了每次切换模型都要手动编辑 settings.json 的痛点。

核心价值：不用记 Base URL，不用手动编辑 JSON 文件，不用重启终端。选择 → 填 Key → 切换，三步搞定。

主要功能一览

🔄 供应商管理

  • 内置 50+ 供应商预设：智谱、DeepSeek、MiniMax、MiMo、通义千问、OpenRouter、SiliconFlow 等
  • 选择供应商后只需填入 API Key 即可使用
  • 支持自定义供应商（手动输入 Base URL）

⚡ 一键切换

  • 主界面或系统托盘秒级切换供应商
  • Claude Code 热重载：无需重启终端即可使用新配置
  • 支持多应用：Claude Code、Codex、Gemini CLI、OpenCode

🔌 MCP 统一管理

  • 跨应用统一管理 MCP 服务器配置
  • 一处编辑，多处同步（Claude Code / Codex / Gemini CLI）

📊 用量统计

  • Token 消耗追踪、成本监控
  • 趋势图表，直观查看使用情况

🛡️ 故障转移

  • 配置多个供应商的故障转移队列
  • 主供应商故障时自动秒级切换到备用供应商

快速上手

1. 启动 CC-Switch

  • 桌面版：从开始菜单或桌面快捷方式启动
  • CLI 版：在终端运行 cc-switch，进入交互式菜单

2. 添加供应商

  1. 打开 CC-Switch 主界面
  2. 从预设列表中选择供应商（如"智谱 GLM"）
  3. 输入你的 API Key
  4. 点击"保存"

3. 切换供应商

  1. 在主界面选中目标供应商
  2. 点击"切换到此供应商"
  3. 或右键系统托盘图标 → 选择供应商 → 秒级切换

4. CLI 版命令

# 交互模式（菜单选择） cc-switch # 列出所有已配置的供应商 cc-switch provider list # 切换到指定供应商 cc-switch provider switch 1 # 添加新供应商 cc-switch provider add # 删除供应商 cc-switch provider delete 2 # MCP 同步 cc-switch mcp sync

如何安装 CC-Switch

桌面版（推荐）

  • Windows：在 1shot-CC 向导中一键安装，或从 GitHub Releases 下载 .msi 文件
  • macOS：brew tap farion1231/ccswitch && brew install --cask cc-switch

CLI 版

npm install -g cc-switch-cli

常见问题

Q：切换供应商后 Claude Code 需要重启吗？
A：不需要。CC-Switch 桌面版支持热重载，Claude Code 会自动使用新配置。如果没生效，重启终端即可。
Q：为什么切换后 Claude Code 报错？
A：检查 API Key 是否正确、Base URL 是否匹配、账户是否有余额。建议先用对应平台的官方测试工具验证 API Key 可用性。
💡 推荐策略：配置 2-3 个供应商，使用故障转移功能。例如：日常用 DeepSeek（便宜），高峰期自动切换到智谱 GLM（稳定）。
