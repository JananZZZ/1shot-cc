最后一步——告诉 CC-Switch 你想用哪家 AI 服务

💡 CC-Switch 是个桌面小工具，藏在屏幕右下角任务栏里（一个小图标）。它的作用就是帮你管理不同的 AI 服务商，你想用哪个就切到哪个。
①
右键点图标

在屏幕右下角任务栏找到 CC-Switch 图标，右键点击它

②
选一家服务商

点「添加供应商」，从列表里选你申请的（比如智谱 GLM）

③
粘贴 API Key

把你申请到的 API Key 粘贴进去保存，就完成了！
