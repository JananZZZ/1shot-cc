🔑 获取 API 密钥完整教程

国内主流 AI 大模型 API 获取指南 — 手把手图文教学

什么是 API Key？API Key 是访问 AI 服务的"通行证"。把它配置到 Claude Code 中，就能通过国内供应商使用 Claude Code，无需科学上网，费用也更便宜。

一、智谱 AI（GLM 系列）

智谱 AI 是国内老牌大模型厂商，提供 GLM Coding Plan 套餐，专为 Claude Code 优化。

注册与获取步骤

  1. 访问 open.bigmodel.cn，点击右上角"注册"
  2. 使用手机号或邮箱完成注册
  3. 完成实名认证（部分模型需要）
  4. 进入 API Key 管理页
  5. 点击"创建新的 API Key"，命名后生成
  6. 立即复制保存，关闭后无法再次查看

💰 省钱技巧：GLM-4.7-Flash 模型完全免费。通过邀请链接注册可获赠 2000 万 Tokens 额度。

二、DeepSeek（深度求索）

DeepSeek 是目前性价比最高的选择，V4-Flash 低至 0.02 元/百万 tokens。

注册与获取步骤

  1. 访问 platform.deepseek.com
  2. 使用手机号注册（国内手机号即可）
  3. 登录后左侧菜单找到 "API Keys"
  4. 点击"创建 API Key"，命名后生成
  5. 立即复制保存
💰 价格参考：V4-Flash 输入 ¥0.02/百万tokens（缓存命中），输出 ¥2/百万tokens。V4-Pro 限时 2.5 折优惠中。

三、MiniMax（海螺 AI）

MiniMax 的 Token Plan 套餐适合有稳定用量需求的开发者。

注册与获取步骤

  1. 访问 platform.minimaxi.com
  2. 使用手机号或邮箱注册
  3. 完成实名认证（姓名 + 身份证号）
  4. 进入"接口密钥"页面，点击"创建密钥"
  5. 复制 API Key，同时记下 Group ID（16 位字符串）

四、小米 MiMo

小米推出的 AI 模型，Pro 版全球排名第八，还有百万亿 Token 激励计划。

注册与获取步骤

  1. 访问 platform.xiaomimimo.com
  2. 使用小米账号登录（需要先注册小米账号）
  3. 进入控制台 → API Keys
  4. 点击"新建 API Key"，复制保存
  5. 可同时申请"百万亿 Token 激励计划"获取免费额度
💰 免费额度：MiMo-V2-Flash 曾多次延长免费试用。另可申请百万亿 Token 激励计划：100t.xiaomimimo.com

五、通义千问 Qwen（阿里云百炼）

阿里云旗下 AI 平台，新用户赠送大量免费额度。

注册与获取步骤

  1. 注册阿里云账号：aliyun.com
  2. 开通百炼服务：bailian.console.aliyun.com
  3. 进入 API-KEY 管理
  4. 创建新的 API-KEY，复制保存
💰 免费额度：新用户赠送最高 7000 万 Tokens 额度（有效期 180 天）。Coding Plan Pro 版 ¥200/月，每月 90,000 次请求。

API Key 安全提醒

⚠️ 重要：
  • 切勿将 API Key 分享给他人或上传到 GitHub
  • API Key 创建后仅显示一次，务必立即保存
  • 如果不慎泄露，立即在平台上删除旧 Key 并创建新 Key
  • 建议定期轮换 API Key（每 3 个月换一次）
