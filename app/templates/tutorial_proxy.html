🌐 科学上网简明指南

什么时候需要、什么时候不需要、以及安全使用建议

⚠️ 先说重点：如果你用国内服务商（智谱 / DeepSeek / MiniMax / MiMo / 通义千问）的 API，完全不需要科学上网。1shot-CC 已经把下载都切到了国内镜像。

我需要科学上网吗？

✅ 不需要

  • 使用国内 AI 服务商的 API — 国内服务器，直连
  • npm 安装包 — 1shot-CC 已自动配置淘宝镜像
  • Node.js / Git 下载 — 从 npmmirror 国内镜像

⚠️ 有时需要

  • 访问 GitHub（下载 CC-Switch、看开源项目）— 有时能直连
  • 使用 Anthropic 官方 API（api.anthropic.com）— 被屏蔽
结论：大多数人配好国内服务商就能正常使用 Claude Code，不需要折腾科学上网。

什么是科学上网？

科学上网是指通过特定工具访问因网络限制而无法直接访问的境外网站。常用方式有 VPN、代理客户端等。

Clash for Windows

Clash 是目前最流行的代理客户端之一，支持规则分流（国内走直连、国外走代理）。

下载安装

  1. 访问 clashmetas.com.cn/clash-for-windows
  2. 下载安装包（.exe），双击安装
  3. 首次启动如被 Windows 拦截，点「更多信息」→「仍要运行」

基本配置

  1. 从你的服务商那拿到「订阅链接」
  2. 打开 Clash → 左侧点 Profiles
  3. 粘贴订阅链接 → 点 Download
  4. 左侧点 Proxies → 选一个节点
  5. 确保 System Proxy 开关打开
💡 分流模式：Clash 默认 Rule（规则）模式——国内网站直连，国外网站走代理。不用手动切换。

选择服务商（机场）

常见平台如魔戒等。选的时候注意：

  • 优先选运营时间长、口碑好的
  • 先月付试用，别一次性买太久
  • 警惕价格过低的「免费」服务
  • 使用独立的支付方式

npm 淘宝镜像 — 不用科学上网的加速方案

如果你只是想让 npm 下载更快，设一下淘宝镜像就行：

# 设置镜像（1shot-CC 已帮你配好了） npm config set registry https://registry.npmmirror.com # 查看当前镜像 npm config get registry

GitHub 访问替代

  • Watt Toolkit（原 Steam++）：免费开源的网络加速工具
  • 修改 hosts：给 GitHub 添加 CDN IP 加速
  • Gitee：国内代码托管，很多项目有镜像

🔒 安全提醒

  • 不要在代理环境下登录银行、支付等敏感账户
  • 使用开源客户端，避免不明来源的「定制版」
  • 不使用时关闭代理客户端
📌 最后再强调一次：Claude Code 用户配好国内 API 就能直接用，不需要科学上网。1shot-CC 都帮你配好了。只有需要访问 GitHub 或官方 API 时才考虑。
