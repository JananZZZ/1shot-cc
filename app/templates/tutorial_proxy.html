🌐 科学上网简明指南

了解科学上网的概念、什么时候需要、以及安全使用建议

⚠️ 免责声明：本教程仅供学习和了解技术知识之用。请遵守当地法律法规，合理使用网络工具。使用国内中转 API（智谱/DeepSeek 等）即可正常使用 Claude Code，不一定需要科学上网。

一、我需要科学上网吗？

✅ 不需要科学上网的情况

  • 使用国内 AI 供应商（智谱 GLM、DeepSeek、MiniMax、MiMo、通义千问）的 API — 国内服务器，直连访问
  • npm 安装包时使用淘宝镜像源（registry.npmmirror.com）
  • Node.js 和 Git 从国内镜像下载（npmmirror.com）

⚠️ 可能需要的情况

  • 访问 GitHub（下载 CC-Switch、查看开源项目）— 有时能直连
  • 使用 Anthropic 官方 API（api.anthropic.com）— 被墙
  • 访问部分国外技术文档和社区
结论：对于大多数 Claude Code 用户，只要配置了国内供应商的 API，完全不需要科学上网。1shot-CC 已将 Node.js、Git 下载源指向国内镜像，npm 自动配置淘宝源。

二、什么是科学上网？

科学上网是指通过特定的网络工具和技术手段，访问因网络限制而无法直接访问的境外网站和服务。常见的技术方式包括 VPN、代理（Proxy）、隧道等。

三、Clash for Windows 安装指南

Clash 是目前最流行的代理客户端之一，支持 Windows/macOS/Linux。

下载与安装

  1. 访问下载站：https://clashmetas.com.cn/clash-for-windows
  2. 下载最新版 Clash for Windows 安装包（.exe 或 .zip）
  3. 双击安装，或解压 zip 直接运行 Clash for Windows.exe
  4. 首次启动可能被 Windows Defender 警告，点击"更多信息"→"仍要运行"

基本配置

  1. 获取订阅链接（从你的服务提供商处获取，也称为"机场"）
  2. 打开 Clash for Windows → 点击左侧 "Profiles"（配置）
  3. 粘贴订阅链接 → 点击 "Download"
  4. 点击左侧 "Proxies"（代理），选择一个节点
  5. 确保 "System Proxy"（系统代理）开关打开
💡 分流模式：Clash 默认使用 Rule（规则）模式，国内网站直连，国外网站走代理。不用手动切换，非常方便。

四、常用服务平台介绍

魔戒

魔戒是国内知名的网络加速服务平台，提供稳定高速的线路。

  • 官网：mojie.me（可能有变，请自行搜索"魔戒"最新地址）
  • 特点：线路稳定、价格透明、支持 Clash
  • 付费方式：按月/按年订阅
⚠️ 选择服务商注意事项：
  • 优先选择运营时间长、口碑好的服务商
  • 不要一次性购买长期套餐，先月付试用
  • 警惕价格过低的"免费"服务，可能窃取数据
  • 使用独立的支付方式，不要绑定主力银行卡

五、npm 淘宝镜像 — 无需代理的替代方案

如果你只是想让 npm 安装更快，完全不需要科学上网，设置淘宝镜像即可：

# 设置淘宝镜像（1shot-CC 已自动配置） npm config set registry https://registry.npmmirror.com # 查看当前镜像源 npm config get registry # 恢复官方源 npm config delete registry

六、GitHub 访问替代方案

如果 GitHub 访问缓慢或无法打开，可以尝试以下方法：

  • Watt Toolkit（原 Steam++）：免费开源的网络加速工具，支持 GitHub 加速
  • FastGitHub：GitHub 镜像站
  • Gitee：国内代码托管平台，很多开源项目有同步镜像
  • 修改 hosts 文件：添加 GitHub 的 CDN IP（需自行搜索最新 IP）

七、安全提示

🔒 安全第一：
  • 不要在代理环境下登录银行、支付等敏感账户
  • 使用开源客户端（如 Clash Verge），避免使用不明来源的"定制版"
  • 定期检查代理客户端的运行状态，不使用时关闭
  • Claude Code 配置了 CLAUDE_CODE_DISABLE_NONESSENTIAL_TRAFFIC=1，可减少不必要的网络请求
📌 总结：对于 Claude Code 用户来说，使用国内供应商 API + npm 淘宝镜像就可以完全正常工作。只有在需要访问 GitHub、使用 Anthropic 官方 API 等特殊情况时才需要科学上网。1shot-CC 已经帮你配置好了所有国内加速方案，开箱即用！
