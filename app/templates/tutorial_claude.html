什么是 Claude Code？

Claude Code 是 Anthropic 推出的一款命令行 AI 编程助手。它在终端（命令窗口）中运行，通过自然语言对话帮你完成代码编写、调试、重构、文件管理等工作。简单说，就是一个懂编程的 AI 助手，住在你的终端里。

核心定位：它不是 IDE 插件，而是一个独立的 CLI（命令行界面）工具。你可以在任何终端中使用它。

基本功能介绍

1. 代码生成与编辑

直接告诉 Claude Code 你想要什么，它会自动创建、编辑文件：

  • 「帮我写一个 Python 爬虫脚本」→ 自动创建文件并写入代码
  • 「把这个函数改成异步的」→ 自动定位并修改代码
  • 「给所有 API 路由加上错误处理」→ 批量修改多个文件

2. 代码理解与导航

Claude Code 能理解整个项目的代码结构：

  • 「这个项目是做什么的？」→ 分析项目结构给出概述
  • 「解释 src/auth/login.ts 的逻辑」→ 深入分析特定文件
  • 「找出所有调用 deleteUser 的地方」→ 代码搜索和引用追踪

3. 调试与修复

遇到 bug 时，直接描述现象：

  • 「运行 npm test 报错了，帮我看看」→ 自动运行并分析错误
  • 「这个 API 返回 500 错误是怎么回事」→ 追踪代码路径找根因

4. Git 操作

Claude Code 可以帮你管理版本：

  • 「帮我提交这些改动，写个好的 commit message」
  • 「创建一个 PR」

5. Shell 命令执行

直接在 Claude Code 中运行终端命令：

  • 「安装 express 和它的类型定义」
  • 「运行数据库迁移」

详细名词解释

术语含义
Agent（代理）具有特定能力的子任务执行单元。Claude Code 可以启动多个 Agent 并行工作，比如一个探索代码库、一个写测试、一个做代码审查
Tool（工具）Claude Code 可以调用的操作接口。包括读写文件（Read/Write/Edit）、运行命令（Bash）、搜索代码（Grep/Glob）、网络请求（WebFetch/WebSearch）等
Hook（钩子）在特定事件触发时自动执行的操作。分为 PreToolUse（工具使用前）、PostToolUse（工具使用后）、Stop（会话结束时）等类型，常用于代码格式化、安全检查等自动化流程
MCP（Model Context Protocol）模型上下文协议。允许 Claude Code 连接外部服务（数据库、API、文件系统等），扩展其能力边界。例如通过 MCP 连接 GitHub、Slack、数据库等
Context（上下文）Claude Code 能"看到"的信息。包括当前对话、项目文件、git 状态等。上下文越大，理解越全面，但会消耗更多 Token
TokenAI 处理文本的基本单位。约 1 个中文字 ≈ 1.5-2 个 Token。API 按 Token 计费
Extended Thinking（扩展思考）Claude Code 的深度推理模式。在复杂问题上会自动"深思熟虑"，得出更准确的答案。可按 Alt+T 切换
Plan Mode（计划模式）在正式修改代码前，先制定详细的实现计划。用户审核通过后再动手实施，适合大型改动
CLAUDE.md项目级配置文件。可以写项目说明、编码规范等，Claude Code 会在每次会话中自动加载
settings.json用户级配置文件（~/.claude/settings.json）。配置 API 端点、模型映射、环境变量等
CC-Switch第三方桌面工具，用于管理多个 API 供应商（智谱、DeepSeek 等），一键切换，50+ 供应商预设

常用快捷键

快捷键功能
Ctrl + C取消当前操作 / 中断 AI 回复
Ctrl + D退出 Claude Code 会话
Ctrl + O查看 AI 的详细思考过程（Extended Thinking）
Alt + T切换 Extended Thinking 模式开关
↑ / ↓浏览历史命令
Tab自动补全文件路径
Ctrl + L清屏
Esc清除当前输入

命令行参数

参数说明
claude以交互模式启动
claude "你的问题"直接提问并退出（非交互模式）
claude --model haiku使用指定模型（haiku/sonnet/opus）
claude --dangerously-skip-permissions跳过所有安全确认（危险，谨慎使用）
claude config打开配置面板
claude update更新到最新版本
claude --version查看当前版本号

常用命令（在 Claude Code 对话中输入）

命令功能
/help查看帮助
/clear清除对话历史，开始新会话
/compact压缩上下文，释放 Token 空间
/config打开配置面板
/cost查看当前会话的 Token 消耗
/doctor诊断环境问题
/review代码审查当前改动
/security-review安全审查
💡 新手建议：刚开始使用时，每次只让 Claude Code 做一件事。等熟悉了它的工作方式后，再尝试更复杂的任务。配置好 CLAUDE.md 可以大大提升使用体验！
⚠️ 注意：使用 --dangerously-skip-permissions 参数会让 Claude Code 跳过所有安全确认，可能导致文件被意外修改。除非你完全信任当前操作，否则不要使用。
