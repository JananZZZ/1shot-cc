Claude Code 是什么？

Claude Code 是一个在命令行（终端）里工作的 AI 编程助手。你不用写代码，直接用大白话告诉他「帮我做什么」，他就会自动完成。

一句话理解：它就是个懂编程的聊天机器人。你跟它说要做啥，它帮你做。

能帮干什么？

写代码

告诉它「帮我写一个 Python 爬虫抓取天气数据」→ 自动创建文件、写完所有代码。

改代码

告诉它「把这个函数改得更快一点」→ 自动定位代码、优化、保存。

找 Bug

告诉它「运行测试报错了，帮我看看为什么」→ 自动运行测试、分析错误、修复。

解释项目

告诉它「这个项目是干嘛的？给我讲讲」→ 分析代码结构，用大白话解释给你听。

管理文件

告诉它「把这三个文件夹合并，删除没用的文件」→ 自动操作文件和 git。

怎么启动？

  1. 打开项目文件夹
  2. Shift + 右键空白处 → 选「在此处打开 PowerShell 窗口」
  3. 在窗口中输入 claude 然后回车
  4. 等几秒加载完成，出现聊天界面
  5. 开始和它对话！

常用快捷键

快捷键功能
Ctrl + C中断当前操作，让它停下来
Ctrl + D退出 Claude Code
Ctrl + O查看它的思考过程（进阶功能）
↑ ↓浏览之前输入过的命令
Tab自动补全文件路径

对话里的命令

命令用途
/help查看帮助
/clear清空对话，重新开始
/cost看看用了多少 Token（花了多少钱）
/doctor检查环境有没有问题
claude update更新到最新版本

名词小词典

名词大白话解释
TokenAI 计算的基本单位。约 1 个汉字 ≈ 1.5 个 Token。按用量收费
AgentClaude Code 可以同时派几个"小助手"分头干活
Hook自动执行的"套路"，比如写完代码自动检查格式
MCP连接外部工具的协议，比如连上数据库、GitHub
CLAUDE.md项目说明书。告诉 Claude Code 你的项目是干嘛的
Plan Mode先出计划再动手。适合大改动，防止改错了
💡 新手建议：刚开始每次只让它做一件事。熟悉之后再给它更复杂的任务。并且可以把你的项目说明写在 CLAUDE.md 里，它会理解得更好。
⚠️ 注意：使用 --dangerously-skip-permissions 参数会跳过所有安全确认，可能意外改错文件。除非你完全信任当前操作，否则不要用。
