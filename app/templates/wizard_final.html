🖥️ 如何启动 Claude Code？

✅ 推荐：PowerShell（界面更好看）

PowerShell 对 Claude Code 的富文本渲染支持更好，排版不容易混乱。

  1. 在项目文件夹中，按住 Shift + 右键空白处
  2. 选择 "在此处打开 PowerShell 窗口"
  3. 或者按 Win + R，输入 powershell，回车
  4. 也可以在开始菜单搜索 PowerShell
  5. 进入项目目录后输入 claude 并回车
💡 技巧：在 PowerShell 中按 Ctrl+Shift++ 可以放大字体，Ctrl+- 缩小。
备选：命令提示符（CMD）

传统的 Windows 命令窗口，Claude Code 可以运行但可能出现排版问题。

  1. 按 Win + R，输入 cmd，回车
  2. 或在文件夹地址栏输入 cmd 回车
  3. 或在开始菜单搜索 cmd 或 命令提示符
  4. 进入项目目录后输入 claude 并回车
⚠️ 注意：CMD 对 Unicode 和富文本支持较差，可能出现字符错位。建议优先使用 PowerShell。
👋 第一次使用 Claude Code
  1. 在终端输入 claude 并回车
  2. Claude Code 会扫描当前文件夹，可能提示"是否信任此文件夹"——选 Yes
  3. 等待几秒加载完成后，会出现对话界面
  4. 试试输入：hello world！帮我写一个简单的 Python 脚本
🎯 第一次对话建议：从一个简单任务开始。比如让 Claude Code 创建一个新文件、解释一段代码、或者帮你运行一条命令。熟悉了基本用法后，再尝试复杂的任务。
