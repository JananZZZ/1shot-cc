✅ 推荐：用 PowerShell（好看）
  1. 打开你的项目文件夹
  2. 按住 Shift + 在空白处点右键
  3. 选「在此处打开 PowerShell 窗口」
  4. 输入 claude → 回车 ✨
💡 按 Ctrl+Shift++ 可以放大字体
备选：用 CMD（传统）
  1. 按 Win+R
  2. 输入 cmd → 回车
  3. 输入 claude → 回车
⚠️ CMD 下排版可能不太好看，建议用 PowerShell
